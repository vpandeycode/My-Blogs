# 🌐 vpandey.codes

**vpandey.codes** is my personal portfolio and blogging website where I share my thoughts, showcase the projects I'm building, and document my journey as a developer. It serves as a central place for anyone interested in my work to learn more about me, explore my projects, and connect with me.

## Why I Built This

I wanted a platform where I could publicly showcase what I'm working on instead of letting my projects remain scattered across different platforms. This website allows me to:

* Share my development journey and personal thoughts through blogs.
* Showcase projects I'm building or actively working on.
* Display creative work, including edited videos.
* Create a space where people can know me better and connect with me.

## Technologies Used

This website is built using:

* **HTML5**
* **CSS3**
* **JavaScript**

### Deployment

The website is deployed using:

* **GitHub**
* **Cloudflare**

## Website Sections

### About

### Projects
